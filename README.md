## 📊 Project Description

This project presents an end-to-end data analysis of Walmart's weekly sales data across 45 stores. The objective was to evaluate overall sales performance, identify top and low performing stores, analyze seasonal sales patterns, and study the impact of external economic factors such as unemployment, CPI, fuel price, and temperature.

The project demonstrates practical business analytics using SQL for querying, Python for data analysis, and Power BI for interactive dashboard visualization.

---

## 🎯 Key Highlights

- Analyzed total sales of ₹6,737M+ across 45 stores
- Identified highest and lowest performing stores
- Evaluated store contribution percentage
- Compared holiday vs non-holiday sales performance
- Performed correlation analysis between sales and external factors
- Built a 3-page interactive Power BI dashboard

---

## 🛠 Technologies Used

- MySQL (Data Storage & SQL Analysis)
- Python (Pandas, Matplotlib)
- Power BI (Dashboard & Visualization)
- Excel (Data Cleaning)

---

## 📈 Business Insights

- Sales peak observed in July.
- Store 20 contributes the highest revenue share.
- Holiday periods show higher average sales.
- Unemployment shows a negative correlation with weekly sales.
- Economic indicators moderately influence consumer purchasing behavior.

---

## 🚀 Outcome

This project showcases the complete data analytics workflow including data cleaning, database management, SQL querying, statistical analysis, and business intelligence reporting.

It highlights the ability to transform raw data into actionable business insights.
